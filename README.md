# Minimal YM EA (t)

Minimal/skeleton Expert Advisor using the YM library for MetaTrader 5.

**Author:** Yousuf Mesalm | www.yousufmesalm.com | WhatsApp +201006179048 | Upwork: https://www.upwork.com/freelancers/youssefmesalm

## Requirements

- MetaTrader 5
- YM library in `MQL5/Include/YM`

## Installation

Copy `t.mq5` into `MQL5/Experts/`.

## Compilation

Compile in MetaEditor (F7). Include path must point to the terminal MQL5 directory (so `Include/YM` is found).

## Usage

Minimal EA; attach to a chart for testing or as a base for custom logic.
